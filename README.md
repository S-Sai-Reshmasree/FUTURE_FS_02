📊 Mini CRM - Client Lead Management System

🚀 Project Overview

This is a Mini Client Lead Management System (CRM) built as part of a Full Stack Web Development internship task.
The system helps manage leads generated from website contact forms by tracking their status, source, and follow-ups.
It simulates how real companies manage potential customers before converting them into clients.

🎯 Objective

To build a simple CRM system that allows users to:

* Store client leads from contact forms
* Track lead status (New → Contacted → Converted)
* Manage and delete leads
* Maintain a simple dashboard UI

 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage (for data persistence)

✨ Features

✔ Add new leads (Name, Email, Source)
✔ View all leads in dashboard
✔ Update lead status:

* New
* Contacted
* Converted
  ✔ Delete leads
  ✔ Data stored in browser (LocalStorage)
  ✔ Responsive and clean UI
